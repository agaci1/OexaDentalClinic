User Scenarios

1. User Login
Actor: User

User enters email and password
System validates credentials
User is logged in

2. User Registration

User enters personal details
System saves account
Confirmation message displayed

3. User Logout

User clicks logout
System ends session
Extended Scenarios
Invalid password → error message
Email already exists → registration blocked
