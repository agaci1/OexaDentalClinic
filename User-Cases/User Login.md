## UC_01 – User Login

| Field | Description |
|------|------------|
| **UC Name** | UC_01 – User Login |
| **Summary** | User logs into the system using credentials |
| **Dependency** | None |
| **Actors** | Primary: User (Patient, Dentist, Manager, Marketer) |
| **Preconditions** | User has a registered account |
| **Main Sequence** | User opens login page; User enters email and password; User clicks login; System validates credentials; System grants access |
| **Alternative Sequence** | If credentials are invalid → system shows error; If password forgotten → user requests reset |
| **Non-functional requirements** | Fast response time; Secure authentication |
| **Postconditions** | User is successfully logged into the system |
