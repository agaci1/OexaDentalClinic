US_01: User registers account
1.	User navigates to registration page 
2.	User enters personal details 
3.	User submits the form 
4.	System validates the data 
5.	System creates account successfully 
6.	If data is invalid: System shows error message 

US_02: User logs in
1.	User navigates to login page 
2.	User enters email and password 
3.	User clicks login 
4.	System validates credentials 
5.	System grants access 
6.	If credentials are incorrect: System displays error message 

US_03: Book appointment
1.	User logs into the system 
2.	User selects a dentist 
3.	User chooses date and time 
4.	System checks availability 
5.	User confirms appointment 
6.	System creates appointment 
7.	System sends notification 

US_04: Book special appointment
1.	User selects special appointment option 
2.	User chooses dentist for children 
3.	User selects time slot 
4.	System checks availability 
5.	System confirms appointment 

US_05: Receive reminder
1.	System monitors upcoming appointments 
2.	System sends reminder email 
3.	User receives notification 


US_06: View offers
1.	User navigates to offers section 
2.	System displays promotions 
3.	User views available discounts 


US_07: View appointment history
1.	User logs into the system 
2.	User opens history section 
3.	System displays past appointments


US_08: Cancel appointment
1.	User selects appointment 
2.	User clicks cancel 
3.	System updates appointment status 
4.	System sends confirmation notification 


US_09: Rate dentist
1.	User opens completed appointment 
2.	User selects rating 
3.	User submits feedback 
4.	System stores review 


US_10: Dentist views schedule
1.	Dentist logs into the system 
2.	Dentist opens schedule 
3.	System displays appointments 

US_11: Mark appointment completed
1.	Dentist selects appointment 
2.	Dentist marks as completed 
3.	System updates appointment status 

US_12: View patient records
1.	Dentist selects a patient 
2.	System retrieves patient data 
3.	System displays medical history 

US_13: Update treatment
1.	Dentist opens patient record 
2.	Dentist adds or updates treatment 
3.	System saves changes 

US_14: Manage users
1.	Manager logs into the system 
2.	Manager opens admin panel 
3.	Manager adds/edits/removes users 
4.	System updates database 
US_15: Assign patient to dentist
1.	Manager selects patient 
2.	Manager selects dentist 
3.	System assigns patient 
4.	System updates records
